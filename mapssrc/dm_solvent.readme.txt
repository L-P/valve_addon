Industrial themed open map for Half-Life: Deathmatch.

Unzip under the `valve_addon` directory and allow custom addon content
in the game settings.

Compiled and lit using [https://ericwa.github.io/ericw-tools/|ericw-tools], built using [https://trenchbroom.github.io/|TrenchBroom].
